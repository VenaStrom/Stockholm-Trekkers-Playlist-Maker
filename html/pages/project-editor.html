Playlists are constructed with blocks. A block is the episodes in between pauses. You only set the start time of the first file in the block and then the rest get queued up so they play immediately after the previous episode. The pauses are inserted automatically at the correct length. This is done to avoid gaps in between content and for the sake of easier creation of playlists.
